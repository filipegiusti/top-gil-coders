Descrição:	Este arquivo padroniza os arquivos objetos, que afetam a saída gerada pelo montador e o que o ligador pode esperar receber na sua entrada.

- Estrutura do arquivo:

1	x INT				 4 bytes	Tamanho do código
1	x INT				 4 bytes	Tamanho da pilha necessária
1	x INT				 4 bytes	Endereço do começo da execução
n	x STRUCT SIMBOLO		13 bytes	Simbolo global
{
	8	x CHAR		 1 byte	Nome do símbolo
	1	x INT			 4 bytes	Endereço do símbolo
	1	x CHAR		 1 byte	Informação de relocação
}
n 	x STRUCT CODIGO_RELOC	34 bytes	Instruções de máquina + sua informação de relocação
{
	16	x SHORT INT		 2 bytes	Instruções de máquina
	1	x SHORT INT		 2 bytes	Informação de relocação
}

- Observações

As STRUCT SIMBOLO acabam quando for encontrado uma com nome, endereço e informação NULL.
A última STRUCT CODIGO_RELOC poderá ter alguns SHORT INT´s que não são efetivamente instruções de máquina, mas estão ali para completar a STRUCT, isso pode ser verificado utilizando o tamanho do código.
